# 🚀 ApplyFlow

A simple React application to keep track of job applications in one place.

Instead of using notes or spreadsheets, ApplyFlow lets you easily add, edit, delete, search and filter your job applications. It also saves your data in Local Storage, so everything stays even after refreshing the page.

## 🌐 Live Demo

- 🔗 **Live Project:** https://manishkumar-365.github.io/apply-flow/

---

## ✨ Features

- ✅ Add a new job application
- ✅ Edit existing applications
- ✅ Delete applications
- ✅ Search applications by company name
- ✅ Filter applications by application status
- ✅ Live dashboard showing:
  - 📄 Total Applications
  - 🟢 Applied
  - 🟡 Interviewed
  - 🔴 Rejected
- ✅ Data stored using Local Storage
- ✅ Responsive design for desktop and mobile
